# Personal-Finance-Dashboard-Power-BI
Power BI dashboard to analyze income, expenses, and savings.

🔍 Overview

This project is a Power BI dashboard designed to track and analyze personal income, expenses, and savings.

📌 Features

Tracks total income, expenses, and savings

Category-wise expense analysis (Food, Rent, Travel, etc.)

Monthly trends using interactive visuals

Slicers for dynamic filtering

🛠️ Tools Used

Power BI

Excel

DAX (basic calculations)

📊 Insights

Identified high spending categories

Analyzed monthly financial trends

Helped improve budgeting decisions

.pbix file (Power BI dashboard)
